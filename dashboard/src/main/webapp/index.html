Getting Started

Boundless Suite offers a variety of ways to import, export, transform, and serve spatial data to the web, and make it interactive.
Find your desired task below and explore the various options.
For more inspiration, visit our website and join the community at Boundless Connect.

1
Prepare Data ...

  • Where to get GIS data:

Natural Earth

Natural Earth has free shapefiles available for download. Try downloading the Natural Earth Quickstart kit for a quick default set.

http://www.naturalearthdata.com

GeoNames

GeoNames has place names (for oceans, all countries, cities, etc.).

You could start with all countries data.

http://www.geonames.org

OpenStreetMap

OpenStreetMap is a freely editable map of the world that can be downloaded in parts.

You can find individual country data to download from Geofabrik.

http://www.openstreetmap.org

2
Load Data ...

  • Choose one of:

Command Line import

Shapefiles can be loaded into PostGIS using the shp2pgsql command-line tool.

shp2pgsql

Tip: If you're working on a remote server you can also download data directly to that server using wget and one of the GIS data download links listed above.

GeoServer

  • You can load data into GeoServer directly:
  • GeoServer Importer
  • If you have raster files, typically they are loaded directly to GeoServer.

Boundless Desktop

Use the desktop client QGIS and the Boundless GeoServer Explorer plugin to load and publish data to GeoServer and PostGIS.

(Boundless Desktop is available separately from Boundless.)

About

Boundless Suite

Boundless Suite powers web, mobile, and desktop applications across the enterprise. Built on leading open source software–including PostGIS, GeoServer, and GeoWebCache–Boundless Suite optimizes standards-based technology for organizations large and small.

Boundless

Boundless is the most trusted provider of enterprise-grade, open source geospatial software. From database to server to client, Boundless reduces the costs of developing, deploying, and maintaining geospatial software. Companies look to Boundless when they want to free themselves of the rigid architectures and unsustainable pricing models of proprietary geospatial software.
