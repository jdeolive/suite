Web SDK QuickView
=================

Web SDK QuickView is a web application providing map composition and editing features. It is intended to demonstrate some of the capabilities of the Web SDK, and is not intended as a fully-featured mapping application.

This section will provide an overview of the features and functionality of the Web SDK Quickiew Demo application.
