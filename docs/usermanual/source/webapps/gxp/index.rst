.. _webapps.gxp:

Application development with Boundless SDK GXP template
=======================================================

This tutorial will discuss how to create an application using the "GXP" template of the Boundless SDK. This template generates a versatile application using GeoExt and Openlayers 2.

The full API documentation for the SDK is available at http://suite.opengeo.org/docs/sdk-api/, and is `included <../../sdk-api>`_ with this documentation bundle.

For more information on the :file:`suite-sdk` script itself, used throughout this tutorial for creating and testing applications, please see the section on :ref:`webapps.sdk`.
