To verify that the extension has been installed properly:

#. Log in to the GeoServer web interface.

#. Click :guilabel:`Stores` then :guilabel:`Add new store`.

#. In the list of :guilabel:`Vector Data Stores`, you should see an entry for :guilabel:`MongoDB`.

   .. figure:: /install/include/ext/img/mongodb_storelink.png

      MongoDB in the list of vector stores

   If you don't see this entry, the extension did not install correctly.
