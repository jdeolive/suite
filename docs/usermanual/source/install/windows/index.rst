Installing Boundless Suite on Windows
=====================================

This section describes how to install or upgrade Boundless Suite on Windows. Please select the type of install below.

.. toctree::
   :maxdepth: 2

   tomcat/index
   vm

.. note:: Installers are no longer available for Windows. To install the latest version of Boundless Suite, you will need to use either the application server-based install (using WARs) or use the :ref:`virtual machine <install.windows.vm>`.

.. warning:: If you need to use PostGIS on Windows, use the :ref:`virtual machine <install.windows.vm>`.
