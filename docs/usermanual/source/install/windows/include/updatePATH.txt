#. Open Start and type :guilabel:`system environment`.

#. Select :guilabel:`Edit the system environment variables`.

   .. figure:: /install/windows/include/edit_system_enviro.png

#. Click :guilabel:`Environment Variables`.

   .. figure:: /install/windows/include/click_enviro_var.png

#. In the :guilabel:`System Variables` section, select :guilabel:`Path`.

#. Click :guilabel:`Edit...`

   .. figure:: /install/windows/include/edit_path.png

#. Add the path:

   * **On Windows 2012 Server**:

     #. Add the new directory (from above) to the beginning of the :guilabel:`Path` :guilabel:`Variable value`, and ensure you end it with a semicolon ('';'').

        .. figure:: /install/windows/include/add_path.png

     #. Click :guilabel:`OK` and then :guilabel:`OK` again.
       
   * **On Windows 10**:     

     #. Click :guilabel:`New`.

     #. Type in the new directory (from above).

        .. figure:: /install/windows/include/win10_edit.png

     #. Click :guilabel:`OK` and then :guilabel:`OK` again.
